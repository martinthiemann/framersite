{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Data types ##### Boolean A boolean can only have two values: **true** or **false**: 
myBoolA = true
myBoolB = false
##### Numbers Numbers can be small, big, rounded, positive or negative: 
myNumberA = 2
myNumberB = 20.0
myNumberC = -223894
You can calculate with numbers like this (*operators*): 
myNumberA + myNumberB  # Add, output: 22.0
myNumberA / myNumberB  # Divide, output: 0.10
myNumberA * myNumberB  # Product, output: 40.0
myNumberA ** myNumberB # Power, output: 400.0
##### String A string is a set of characters like words or complete sentences: 
myStringA = "Hello there"
Strings can be combined: 
myStringB = "#{myStringA}, how are you?"
# Output: "Hello there, how are you?"
Strings can also hold more complex values: 
myStringC = "rgba(255,0,0,0.1)"
myStringD = "1px solid #FFFFFF"
##### Array Arrays contain an ordered set of values. The values can be any type. They can be nested (an array with arrays, for instance). 
# An array with only numbers
myListA = [1, 2, 3, 4, 5]

# An array with numbers, a string and another array
myListB = [1, "test", 22, [1, 2, 3]]
Any item in an array can be accessed by its index, which starts at 0: 
myListA[0] # Output: 1
myListA[4] # Output: 5
You can also count the items in an array: 
myListB.length # Output: 4
Or grab a range from the array: 
myListA[0..2] # Output: [1, 2, 3]
##### Object Object properties are accesssed by name, much like a dictionary from a phone book. The 'lookup string' is called a key. 
myObjectA = {name:"Framer", age:12}
Objects can also be defined with indenting instead of parentheses. 
myObjectA =
	name: "Framer"
	age: 12
To find a property value, use the key: 
myObjectA["name"] # Output: "Framer"
An object's properties can be added and removed: 
# Add a property
myObjectB["platform"] = "Mac"

# Remove a property
delete myObjectB["platform"]
